## Running source
1. clone the repository:
```sh
    git clone https://github.com/BiancaNCoelho/RestaUm.git
    cd RestaUm
```
2. install dependencies:
```sh
    pip install -r requirements.txt
```
3. building:
```sh
    python main.py
```
